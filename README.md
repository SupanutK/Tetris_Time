# Tetris_Time
## Details
Tile-matching puzzle game.
## How to play
- "A" Move left.
- "D" Move right.
- "Q" Rotate anticlockwise.
- "E" Rotate clockwise.
- "S" Drop.
- "P" Pause game.
- "O" Restart game.
